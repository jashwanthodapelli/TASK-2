# 🚀Contact Form & To-Do Web App

This is a responsive and interactive web application built using HTML, CSS, and JavaScript.  
This project was developed as part of my Web Development Internship Task-2.

## ✨ Features

### 📩 Contact Form
- Full Name validation
  - Only letters and spaces allowed
- Email format validation
- 10-digit phone number validation
- Message validation (minimum 5 characters)
- Error messages for invalid inputs
- Redirects to To-Do page after successful submission

### 📋 To-Do Application
- Add tasks dynamically
- Delete tasks
- Mark tasks as completed
- Live task statistics
  - Total Tasks
  - Completed Tasks
- In-page notification acknowledgements
- Responsive modern UI
- Keyboard support (Enter key)

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## 📱 Responsive Design
The application is fully responsive and works smoothly on:
- Desktop
- Tablet
- Mobile Devices
